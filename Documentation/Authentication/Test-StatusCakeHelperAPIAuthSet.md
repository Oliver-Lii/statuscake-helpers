# Test-StatusCakeHelperAPIAuthSet

## SYNOPSIS
Tests whether StatusCake API Credentials have been configured

## SYNTAX

```
Test-StatusCakeHelperAPIAuthSet
```

## DESCRIPTION
Returns a boolean value depending on whether StatusCake API Credentials file for the module exists

### EXAMPLE 1
